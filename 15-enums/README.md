# Sección 15 — Enums

> Los **enums** (enumeraciones) son tipos especiales en Java que representan un conjunto fijo de constantes.
> Son ideales para modelar valores que no cambian, como días de la semana, estados, niveles o
> categorías. A diferencia de las constantes `static final`, los enums ofrecen seguridad de tipos,
> pueden tener atributos, constructores y métodos propios.
