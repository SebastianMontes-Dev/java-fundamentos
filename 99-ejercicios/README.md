# Ejercicios de Java — Practica y Refuerza

Bienvenido a la sección de ejercicios. Aquí pondrás a prueba TODO lo aprendido en las 33 secciones del curso. Los ejercicios están organizados por nivel de dificultad y cada uno indica qué conceptos necesitas dominar para resolverlo.

## Cómo usar esta sección

1. **Lee el enunciado** en [`EJERCICIOS.md`](./EJERCICIOS.md)
2. **Intenta resolverlo por tu cuenta** — no mires la solución todavía
3. **Compara tu solución** con la de [`soluciones/`](./soluciones/)
4. **Aprende de las diferencias** — a veces hay múltiples formas correctas
